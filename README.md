# Portfolio — Abetnego Kristiawan Takaredase

This is a Next.js portfolio website built as part of a technical test for **PT BULLS EYE**.

## About

A personal portfolio showcasing my background, skills, and selected web development projects. Built with Next.js 14 (App Router) and plain CSS — no UI framework.

## Pages

- **Home** — Introduction and links
- **About** — Bio, skills, and work experience
- **Portfolio** — Selected projects with detail pages
- **Contact** — Contact information

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- CSS (no Tailwind)
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Build

```bash
npm run build
npm start
```
